# Trademark policy — ÖppenBokföring / Varumärkespolicy

Status: Maintainer draft — **not legal advice** / **inte juridisk rådgivning**

Official project: **https://github.com/vomba/oppenbokforing**

## English

### Purpose

The **ÖppenBokföring** name and logo identify the official open-source project and releases published by the maintainers under **vomba**. The codebase is licensed under **AGPL-3.0-or-later**; this document covers **brand** use, which copyright licenses do not control.

### Allowed without permission

- Truthful references (“built on ÖppenBokföring”)
- Forking the code under AGPL with **a different product name and branding**
- Academic or press mention

### Requires permission

- Using **ÖppenBokföring** (or confusingly similar spelling) as the product name for a distributed app
- Using official logos/icons to imply endorsement or official status
- Domains or app store listings that suggest you are the canonical project

### AGPL relationship

AGPL requires source for distributed modified versions. It does **not** grant trademark rights. A compliant AGPL fork must use its own name unless explicitly licensed.

## Svenska

### Syfte

Namnet och logotypen **ÖppenBokföring** identifierar det officiella öppna källkodsprojektet och utgåvor som publiceras av maintainers under **vomba**. Källkoden licensieras under **AGPL-3.0-or-later**; detta dokument gäller **varumärkes**användning, som upphovsrättslicenser inte reglerar.

### Tillåtet utan tillstånd

- Sanningsenliga hänvisningar (”byggd på ÖppenBokföring”)
- Fork av koden under AGPL med **annat produktnamn och varumärke**
- Akademisk eller pressnämning

### Kräver tillstånd

- Att använda **ÖppenBokföring** (eller förväxlingsbart liknande stavning) som produktnamn för en distribuerad app
- Officiella logotyper/ikoner som antyder stöd eller officiell status
- Domäner eller appbutiker som antyder att du är det kanoniska projektet

### Förhållande till AGPL

AGPL kräver källkod vid distribution av ändrade versioner. Den ger **inte** varumärkesrätt. En AGPL-fork som följer licensen ska använda eget namn om inte annat uttryckligen tillåts.
